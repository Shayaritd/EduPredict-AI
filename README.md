## 📌 Overview

**EduPredict AI** is a Machine Learning-based student performance prediction system that analyzes various student-related factors and predicts academic outcomes.

The project uses data preprocessing techniques, machine learning algorithms, and a Flask web application to provide real-time predictions through an interactive user interface.

The goal of this project is to help educators and institutions understand the factors affecting student performance and make data-driven decisions.

---

# 🚀 Features

✨ **Student Performance Prediction**
- Predict student academic performance using trained ML models.

✨ **Machine Learning Pipeline**
- Complete workflow from data ingestion to model prediction.

✨ **Data Preprocessing**
- Handles missing values, categorical encoding, and feature transformation.

✨ **Multiple ML Algorithms**
- Compares different models and selects the best-performing model.

✨ **Flask Web Application**
- Simple and interactive interface for entering student details.

✨ **Real-Time Prediction**
- Generates predictions instantly based on user inputs.
